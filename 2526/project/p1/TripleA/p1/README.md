<p>
  Beautiful Soup is a Python library that is used for web scraping. It allows you to parse HTML or XML documents into a readable tree-like structure and extract data based on the document’s elements and tags. With Beautiful Soup, users can easily navigate through webpages, search for specific classes or tags, and retrieve important information such as text, links, and attributes. It is commonly used together with libraries such as requests and pandas to automate data collection from websites.

The website that we will be using is from Maukerja. (https://www.maukerja.my/)

This website is a popular job portal in Malaysia that provides job listings and career-related resources for job seekers. It contains various job advertisements from different industries, including information such as job titles, company names, locations, salary ranges, and job descriptions. Besides job vacancies, the website also offers career advice, resume tips, and employment-related articles. The platform is widely used by individuals searching for internships, part-time jobs, and full-time employment opportunities in Malaysia.

We plan to obtain data from the website by extracting one of its job listing sections. By analyzing the website’s HTML code, we will identify the relevant tags and classes that contain the job information. We will then utilize the pandas library and the BeautifulSoup package to extract the information in HTML format. Finally, we will convert the obtained data into a CSV file. In summary, we will efficiently retrieve job listings and related information from the Maukerja website using these tools.
