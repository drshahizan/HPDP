# Data Folder

This folder contains the datasets used in **Project 1: Optimizing High Performance Data Processing for Large Scale Web Crawlers**.

---

## Note on File Size

The dataset files (`raw_data.csv` and `cleaned_data.csv`) exceed GitHub's file size limit and are therefore **not hosted directly in this repository**.

All data files are available via Google Drive using the link below.
