## Project Description

This project aims to develop a high-performance web crawler capable of collecting large-scale structured data from SoyaCincau, a Malaysian technology news website. The collected data will be processed, cleaned, and optimized using high-performance computing techniques such as multithreading, multiprocessing, and distributed processing.

The project consists of two phases:

### Phase 1: Data Collection & Processing

* Develop a web crawler
* Extract structured article data
* Store data in JSON/CSV format
* Clean and preprocess collected data

### Phase 2: High-Performance Optimization

* Implement multithreading
* Implement multiprocessing
* Apply distributed processing (Dask/Spark)
* Compare baseline and optimized performance
* Evaluate throughput, CPU usage, memory usage, and execution time
